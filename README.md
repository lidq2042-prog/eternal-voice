# 念

> 让思念有声，让爱永存 💝

## ✨ 功能特色

- 📱 模拟亲人聊天 - 用亲人的语气和你对话
- 🎤 声音克隆 - 学习并模仿亲人的声音
- 💬 聊天记录导入 - 从微信导入历史消息
- 🌸 温暖界面 - 用心设计的治愈风格

## 📥 快速开始

### 方法一：下载编译好的APK（推荐）

1. 点击上方的 "Actions" 标签
2. 选择最新的编译任务
3. 下载 "eternal-voice-apk"
4. 安装到手机

### 方法二：自己编译

1. 安装 [Flutter](https://flutter.dev/docs/get-started/install)
2. 克隆本项目
3. 运行 `flutter pub get`
4. 运行 `flutter build apk`

## 📖 使用指南

1. 打开App，点击"添加亲人"
2. 输入亲人的称呼和关系
3. 从微信导入聊天记录（可选）
4. 导入亲人的语音（可选）
5. 开始与"亲人"聊天

## 🛡️ 隐私声明

- 所有数据保存在您的手机本地
- 不会上传任何个人信息
- 声音模型仅在您的设备上训练和使用
